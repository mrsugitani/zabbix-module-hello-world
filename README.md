Project Overview
This module is a custom extension for the Zabbix frontend. Its primary purpose is to demonstrate the integration of a new functional page into the Zabbix UI framework.

Features
・Custom Navigation: Adds a "Hello World" menu item to the main Zabbix sidebar.
・MVC Implementation: Uses the Zabbix standard Module framework with dedicated Actions and Views.
・Responsive Layout: Includes a custom-styled view with proper padding and background styling for better visibility.
